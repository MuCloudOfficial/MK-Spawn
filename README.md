#### 插件概述  
__一个修复 Essentials 关于地标系统的问题修复与扩充__  

#### 主要功能 
- 修复 Essentials 的 中文地标 问题  
- 对 Essentials 扩充了一些需求地标（金钱，权限，物品）  

#### 命令  
- /exaw info  ---  显示插件主页  
- /warps      ---  显示已有地标(与Essentials中命令相同，但对管理员提供了显示需求地标的实现)
